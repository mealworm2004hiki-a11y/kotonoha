# 言の葉

読書中に出会った知らない言葉を調べて集める、自分専用の単語帳PWA。

- 言葉を入力(または本のページを撮影)すると、Claude AIが意味・読み・用例を返します
- 調べた言葉は「今読んでいる本」に紐づけて自動で単語帳に記録されます
- 単語帳は本ごとに絞り込み・検索でき、フラッシュカードで復習できます
- データはすべて端末内(IndexedDB)に保存。設定画面からJSONで書き出し/読み込みできます
- 昼は「紙の辞書」、夜は「読書灯」テーマに自動で切り替わります

## 使い始めるまで

### 1. Claude APIキーを取得する

1. [console.anthropic.com](https://console.anthropic.com/) でアカウントを作成
2. クレジットを購入(最低 $5。単語1回の照会は1円未満なので長くもちます)
3. Settings → API keys でキーを作成してコピー

### 2. GitHub Pagesで公開する

1. GitHubで新しいリポジトリを作成(例: `kotonoha`)
2. このフォルダの中身をすべてpush
3. リポジトリの Settings → Pages → Source を「Deploy from a branch」にして
   `main` ブランチ / `/ (root)` を選択して保存
4. 数分後 `https://<ユーザー名>.github.io/kotonoha/` で公開されます

### 3. スマホのホーム画面に追加する

- **iPhone**: Safariで上のURLを開く → 共有ボタン → 「ホーム画面に追加」
- **Android**: Chromeで開く → メニュー → 「ホーム画面に追加」

### 4. APIキーを設定する

アプリを開き、右上の設定アイコンからAPIキーを貼り付けて保存すれば完了です。
キーはこの端末の中(localStorage)にだけ保存され、外部には送られません
(照会時にAnthropicのAPIへ直接送信されるだけです)。

## ローカルで試す

```sh
npx serve kotonoha
```

カメラ機能はHTTPS(またはlocalhost)でのみ動作します。

## バックアップ

設定 → バックアップ → 「書き出す」でJSONファイルが保存されます。
機種変更の際は新しい端末で「読み込む」を押してこのファイルを選ぶだけです。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面(調べる / 単語帳 / 復習 / 設定) |
| `style.css` | 昼「紙の辞書」/ 夜「読書灯」の2テーマ |
| `app.js` | 検索・記録・復習・バックアップの全ロジック |
| `sw.js` | オフライン対応(単語帳の閲覧・復習はオフラインでも可) |
| `manifest.webmanifest` | ホーム画面アプリ化の設定 |
